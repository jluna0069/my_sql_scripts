Asunto: Contadores de performance

Buen día, Marcelo.

Paso a comentarte que, con el fin de poder recolectar estadísticas en forma periódica, nos gustaría agregar algunos contadores de monitoreo a los servidores de bases de datos SQL, a través de la herramienta “Performance”, que viene en Windows. De estos contadores que quisiéramos configurar, un grupo monitorearía el comportamiento del motor SQL y otro grupo haría lo mismo con el Sistema Operativo.

 

Los contadores serían:

 

Sistema Operativo Windows

Contadores de performance

  • Processor\%Processor time (_total)
  • System\Processor queue length
  • Memory\Available Mb
  • Memory\Pages/sec
  • Physical disk\Avg. disk queue length (unidad) (agregar uno por cada instancia de disco existente)

 

 

MS SQL Server

Contadores de performance:

  • SQLServer:Buffer Manager\Buffer Cache Hit Ratio
  • SQLServer:Plan Cache\Cache hit ratio [_total]
  • SQLServer:Database\Log flush waits/sec [_total]

 

 

Desde ya, aguardamos tus comentarios al respecto.

Saludos.
